Hyperlink

Erzeugt einen Hyperlink, der mit WWW-, lokalen Seiten, e-Mail - Adressen oder Ankern verknüpft ist.

 Allgemein
 Tag:  <a>
 Icon:  
 XHTML-Version:  1.0
 HTML-Version:  2.0
 Shortcut:  Strg+H

 Attribute
 href="WWW-Adresse, e-Mail, Datei oder Anker"  Ziel 
 target="Name des Zielframes (s.u.)"  Zielframe 

 Mögliche Werte für das "target=" - Attribut
 _self  Zeigt das Ziel im gleichen Frame an
 _blank  Zeigt das Ziel in einem neuen Fenster an
 _parent  Zeigt das Ziel im vorherigen Fenster-Zustand
 _top  Zeigt das Ziel im gesamten Fenster an und zerstört alle Frames

 Beispiel
<a href="http://www.teamone.de/selfhtml" target="LinkesFrame">Zur SelfHTML-Homepage</a>
