Mit Hilfe dieses Dialogs lassen sich einfach nützliche JavaScripts® in ein Dokument einbinden. Achten Sie bitte darauf, daß die Option  Tags automatisch positionieren  in den Optionen aktiviert ist.

 Highlight-Grafiken 
Dieses Script ändert das Aussehen einer Grafik, sobald sich der Mauszeiger im Bereich der Grafik befindet. Außerdem ist die Grafik mit einem » Hyperlink unterlegt.

 Link-Auswahl 
Es wird eine Auswahlliste angezeigt, die verschiedene Verweise enthält. Sobald der Besucher ein Ziel auswählt, wird der dazugehörige Hyperlink ausgeführt.

 Ticker

 Laufschrift 
In der Statuszeile des Browser (meist links unten) wird eine Laufschrift angezeigt.
Stammt ebenfalls ursprünglich von » Christian Künz und wurde leicht geändert.

 Beispiel
 Siehe Statuszeile


Wenn Sie die beiden Scripts "Ticker" und "Laufschrift" innerhalb des selben Dokuments verwenden, müssen Sie im <body> - Tag die beiden Attribute
onLoad="ticker_scroll();" onLoad="scrolltext();"
zu
onLoad="ticker_scroll(); scrolltext();"
zusammenfassen.
