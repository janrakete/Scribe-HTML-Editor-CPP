Radiobutton

Definiert einen Radiobutton, der die Auswahl einer Option von mehreren Optionen ermöglicht.

 Allgemein
 Tag:  <input type="radio" />
 Icon:  
 XHTML-Version:  1.0
 HTML-Version:  2.0

 Attribute
 value="Interner Absendewert"  Absendewert 
 name="Name des Radiobuttons"  Name 
 checked="checked"  Status 

 Beispiel
<form>Zahlungsweise:<input type="radio" name="bezahlung" value="bar" />Bar<input type="radio" name="bezahlung" value="karte" />Mit Karte</form>


Nur wenn mehrere Radiobuttons den gleichen Namen haben, werden diese in einer Gruppe zusammengefasst und es kann auch wirklich nur eine Option ausgewählt werden.
