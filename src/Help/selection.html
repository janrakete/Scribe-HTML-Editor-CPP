Markierung

 Als Kommentar 

 Beschreibung:  Der Text wird zwischen die Tags <!-- und //--> gesetzt und gilt nun als Kommentar. Dieser Text wird vom Browser nicht beachtet.
 Zeilenumbrüche anhängen 
 Beschreibung:  Fügt jeder Zeile des Textes einen einfachen Zeilenumbruch (<br /> bzw. <br>) an.
 In Tabelle umwandeln... 
 Beschreibung:  Die Textteile, die durch das festgelegte Trennzeichen voneinander getrennt sind, werden in eine Tabelle umgewandelt.

 In Liste umwandeln 
 Beschreibung:  Wandelt die Textzeilen in eine geordnete Liste um.
 In Grossbuchstaben 
 Beschreibung:  Konvertiert den Text in Grossbuchstaben.

 In Kleinbuchstaben 
 Beschreibung:  Konvertiert den Text in Kleinbuchstaben.
 Sonderzeichen umwandeln 
 Beschreibung:  Wandelt alle Sonderzeichen in (X)HTML - Code um (ü wird zum Beispiel zu &uuml;).
 Alle Tags entfernen 
 Beschreibung:  Aus dem Text werden alle Tags entfernt.
