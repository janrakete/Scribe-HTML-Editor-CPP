Von HTML zu XHTML

Glücklicherweise besitzt Scribe! unter  Scribe! - Tools/XHTML - Konverter...  ein Werkzeug, daß Ihnen den größten Teil der Umwandlung abnimmt. Trotzdem muß das Dokument noch einmal per Mensch überprüft werden.

Der Konverter nimmt folgende Änderungen am Quelltext vor:

  • Tags werden XHTML-konform gemacht (» Unterschiede zu HTML)
  • Alle Sonderzeichen werden in XHTML - Code umgewandelt
  • Der <title> - Tag wird an die richtige Position gesetzt
  • Listen- und Auswahleintr‰ge können komplettiert werden
  • Die DTD sowie die XML - Version werden angepasst
Nach der automatischen Konvertierung sind Sie gefragt. Unter » http://validator.w3.org/file-upload.html können Sie überprüfen, ob das Dokument nun wirklich schon XHTML-konform ist. Wenn nein, so werden die Fehler angezeigt. Beliebte Fehler sind das Vergessen von abschließenden Tags (passiert häufig beim <p> - Tag) und Stellungsfehler von Tags.
Als Letztes sollte noch sichergestellt werden, daß der XHTML - Modus von Scribe! aktiviert ist. Diese Option befindet sich unter  Scribe! - Tools/Optionen... .


Unterschiede der Tags in HTML und XHTML
  • Alle Tags und deren Attribute müssen von nun an kleingeschrieben werden
  • Hat ein Tag keinen abschließenden Tag (zum Beispiel bei <br> oder <img>), wird ein Schrägstrich vor die hintere spitze Klammer gesetzt
  • Tags, zu denen ein abschließender Tag gehört, müssen mit diesem auch abgeschlossen werden
  • Die Wertzuweisungen der Attribute müssen von Anführungszeichen eingeschlossen sein
  • Attribute ohne Wertzuweisungen (zum Beispiel noshade oder readonly) existieren in XHTML nicht, auch ihnen muﬂ ein Wert zugewiesen werden
