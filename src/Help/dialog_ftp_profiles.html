FTP - Profile

Sie können mehrere Profile für das "Direkte FTP" anlegen. In jedem Profil werden Daten wie Host- und Benutzername oder Passwort gespeichert.

Wenn Sie einen Internet - Anschluss besitzen, erhalten Sie meist auch gleichzeitig die Möglichkeit, Ihre Homepage auf einem Server zu speichern. Wie so etwas geht, wird in den Anleitungen beschrieben, die Sie von Ihrem Provider (z.B. "T-Online" oder "AOL") erhalten haben sollten.

Um ein Profil anzulegen, gehen Sie wie folgt vor:

  • Geben Sie einen  Profilnamen  an.
  • Bestimmen Sie den  Host-  und  Benutzernamen .
  • Definieren Sie wahlweise ein  Passwort .
  • Geben Sie bei Bedarf noch weitere Daten an.
  • Mit  "Speichern"  steht das Profil in der » Verwaltung zur Auswahl.
Das Passwort wird zwar verschlüsselt gespeichert, doch auch das bietet nicht unbedingt Schutz vor Trojanern. Wenn Sie kein Passwort angeben, wird es bei jedem Verbindungsaufbau erfragt.

Über das Kontext - Menü (rechte Maustaste) der Profile - Liste ist die Funktion  Löschen  abrufbar.
