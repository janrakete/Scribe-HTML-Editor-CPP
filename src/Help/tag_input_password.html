Passwortfeld

Erzeugt ein Passwortfeld zur Eingabe von z.B. Geheimnummern.

 Allgemein
 Tag:  <input type="password" />
 Icon:  
 XHTML-Version:  1.0
 HTML-Version:  3.2

 Attribute
 value="Vorgabewert"  Text 
 name="Name des Passwortfeldes"  Name 
 size="Optische Länge des Feldes in Zeichen"  Größe 
 maxlength="Maximale Länge des Feldes in Zeichen"  Maximale Länge 
 readonly="readonly"  Status 

 Beispiel
<form><input type="password" name="kreditnr" size="10" maxlength="10" /></form>
