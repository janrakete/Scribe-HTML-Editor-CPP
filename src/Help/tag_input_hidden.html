Verstecktes Feld

Erzeugt ein Eingabefeld, das nicht sichtbar in einem Dokument plaziert werden kann. Es ist besonders beim Einsatz von JavaScripts® nützlich.

 Allgemein
 Tag:  <input type="hidden" />
 Icon:  
 XHTML-Version:  1.0
 HTML-Version:  3.2

 Attribute
 value="Vorgabewert"  Text 
 name="Name des Feldes"  Name 

 Beispiel
<form><input type="hidden" value="Nicht sichtbar?" /></form>
