Checkbox

Definiert eine Checkbox, die das Ankreuzen verschiedener Optionen ermöglicht.

 Allgemein
 Tag:  <input type="checkbox" />
 Icon:  
 XHTML-Version:  1.0
 HTML-Version:  2.0

 Attribute
 value="Interner Absendewert"  Absendewert 
 name="Name der Checkbox"  Name 
 checked="checked"  Status 

 Beispiel
<form>Bestellschein:<input type="checkbox" name="artikel1" value="cdplayer" />CD-Player<input type="checkbox" name="artikel2" value="boxen" />Boxen</form>
