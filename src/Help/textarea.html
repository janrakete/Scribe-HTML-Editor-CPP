Textfeld

Das Textfeld dient der Bearbeitung beliebiger (X)HTML - Dokumente sowie anderer Textdateien.
Dateien, die über  Datei/Öffnen...  oder durch die » Verwaltung geöffnet wurden, werden hier angezeigt.
Das Kontext - Menü (rechte Maustaste) stellt verschiedene Funktionen, die (X)HTML betreffen, bereit:

 Tag bearbeiten... 
 Beschreibung:  Der Tag, in dem sich der Cursor befindet, kann über das jeweilige Dialogfenster nachbearbeitet werden.
 Shortcut:  Strg+F3

 Tag entfernen 
 Beschreibung:  Der Tag, in dem sich der Cursor befindet, und sein Endtag werden gelöscht. Dabei spielt es keine Rolle, ob nun der Tag oder der Endtag angeklickt wurde.
 Shortcut:  Strg+Entf
  Style - Sheets 
 Beschreibung:  Cascading Style-Sheet - Befehle werden an das Ende des aktiven Tags angehängt.

  JavaScript® - Ereignisse 
 Beschreibung:  JavaScript-Ereignisse werden an das Ende des aktiven Tags angehängt.
  Stichwort... 
 Beschreibung:  Zum aktiven Tag wird das betreffende Hilfe - Dokument aufgerufen.
 Shortcut:  F1
  Ausschneiden 
 Beschreibung:  Schneidet den markierten Text aus und kopiert ihn in die Zwischenablage.
 Shortcut:  Strg+X

  Kopieren 
 Beschreibung:  Kopiert den markierten Text in die Zwischenablage.
 Shortcut:  Strg+C

  Einfügen 
 Beschreibung:  Fügt einen Text aus der Zwischenablage ein.
 Shortcut:  Strg+V

 Löschen 
 Beschreibung:  Löscht den markierten Text.
 Markierung 
 Beschreibung:  » Verschiedene Funktionen, die den markierten Text betreffen.
 Schließen 
 Beschreibung:  Schließt das aktuelle Dokument.
 Shortcuts im Textfeld
 Strg+Enter  Fügt <br /> ein.
 Shift+Enter  Fügt <p> und </p> ein.
 Strg+Leertaste  Fügt &nbsp; ein.
 Strg+"  Fügt &quot; ein.
 Strg+<  Fügt &lt; ein.
 Strg+>  Fügt &gt; ein.


Lesezeichen im Textfeld können mit Strg+Umsch+Zahl gesetzt werden. Mit Strg+Zahl können sie wieder abgerufen werden. Schalten Sie dazu bitte die Ansicht der Zeilennummerierung ( Ansicht/Zeilennummerierung ) ein.
