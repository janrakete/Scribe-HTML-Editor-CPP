Projekt - Backup

Mit Hilfe des Projekt - Backups können Sie ihre Projekte schnell und bequem sichern. Die so erstellten Archive lassen sich per e-Mail verschicken oder auf einen anderen Datenträger kopieren. Gesicherte Projekte können natürlich auch wiederhergestellt werden.

 Projekt sichern 

 Projektname 
Wählen Sie hier das zu sichernde Projekt aus.

 Zielverzeichnis 
Bestimmen Sie das Verzeichnis, in dem das gesicherte Projekt gespeichert werden soll.

 Projekt wiederherstellen 

 Archiv 
Wählen Sie hier ein bereits vorhandenes Projekt - Backup aus (Endung: *.cab). Sobald die Wiederherstellung abgeschlossen ist, wird das Projekt in der » Verwaltung angezeigt.

Die Projekte werden im CAB - Format zusammengefasst. Dieses Format ist von den meisten Ent-/Packern lesbar.
