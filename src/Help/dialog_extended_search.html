Erweitertes Suchen & Ersetzen

Dieses Tool ermöglicht das Suchen bzw. Ersetzen bestimmter Textteile in mehreren Dateien gleichzeitig.

Geben Sie einen Suchtext ein ( Suchen nach ) und bestimmen Sie, in welchen Dateien gesucht werden soll ( Suchen in ). Klicken Sie dann auf  "OK" .

Sobald der Vorgang abgeschlossen ist, können Sie sich die gefundenen Textpassagen durch einen Doppelklick in der Ergebnisliste anzeigen lassen.

Der Suchtext kann auch mehrzeilig sein.
