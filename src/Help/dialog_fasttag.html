FastTag

FastTag ermöglicht es, für jeden Button, über den ein Tag - Dialog geöffnet werden kann, eine 2. Belegung zu definieren. Damit können die Dialoge umgangen werden. Diese Funktion ist vor allem für Profis interessant.

Die eigene Definition wird über das Drücken der Strg - Taste und dem gleichzeitigen Klicken auf den entsprechenden Button in das » Textfeld eingefügt.

Probieren Sie es am Besten mal eben aus: Halten Sie Strg - Taste gedrückt und klicken Sie mit der linken Maustaste auf . Anstatt des gewohnten Dialogs wird direkt ein Tag eingefügt.
