Plugins

Plugins stellen neue Funktionen für Scribe! bereit. Sie können auf den Text im Editor zugreifen und ihn verändert zurückgeben.

Plugins stammen meistens von Fremdanbietern. Eine Liste der gemeldeten Plugins existiert auf der » Scribe! - Homepage.
Dort finden Sie auch eine Schnittstellen - Beschreibung, um eigene Plugins mit Delphi® oder C++Builder® zu programmieren. Die Scribe! - Schnittstelle orientiert sich an den Definitionen einer » gemeinsamen Plugin - Schnittstelle für Web - Editoren.

Mit  "Ausführen"  oder einem Doppelklick starten Sie das gewählte Plugin.

Plugins des » HTML Editors von Ulli Meybohm können ebenso mit Scribe! verwendet werden.
