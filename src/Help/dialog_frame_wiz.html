Frame - Assistent

Der Frame - Assistent dient zur Einteilung eines Dokuments in verschiedene Bereiche. Dabei wird in jedem Teil - Bereich (Frame) ein Internet - Dokument angezeigt. So kann zum Beispiel auf der linken Seite eine Navigationsleiste angezeigt werden, und auf der rechten Seite der dazu passende Inhalt.

Der Frame - Assistent stellt die gängigsten Einteilungen zur Verfügung. Die » Eigenschaften der einzelnen Frames können dann über die nachfolgenden Registerreiter eingestellt werden.
