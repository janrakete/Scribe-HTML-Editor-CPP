Neue Verknüpfung

Dieses Tool dient dazu, über Scribe! andere Anwendungen auszuführen. Das können zum Beispiel weitere Editoren oder Grafikprogramme sein.

Sie können bis zu 5 Anwendungen definieren. Gehen Sie dazu wie folgt vor:

  • Wählen Sie zunächst einen freien Platz (1 bis 5) aus.
  • Bestimmen Sie dann die Befehlszeile, die Bezeichnung und die Parameter (siehe unten).
  • Klicken Sie auf  "Speichern" .
  • Wenn Sie den Dialog nun schließen und auf klicken, wird die neue Anwendung aufgelistet sein.
Wenn Sie die Variable "#File" als einen der Parameter verwenden, wird die Variable durch den Pfad und den Dateinamen des aktuellen Dokuments ersetzt.
