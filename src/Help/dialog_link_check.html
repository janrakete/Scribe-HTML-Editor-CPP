Die Link - Überprüfung sucht nach Verweisen auf Dateien wie zum Beispiel » Grafiken, » Multimedia - Objekten und anderen Dokumenten. Links auf externe Seiten im Internet können wahlweise auch getestet werden.

Durch einen Doppelklick auf einen überprüften Link im Dialogfenster wird der betreffende Tag im Quelltext markiert.

Achtung: Verweise auf Ordner im Internet wie z.B. » http://www.scribe.de/deutsch/ oder auch » http://www.scribe.de/ werden leider nicht akzeptiert. Es muß explizit auf einen Dateinamen verwiesen werden.
