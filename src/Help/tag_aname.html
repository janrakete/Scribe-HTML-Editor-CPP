Anker

Legt einen Anker fest, der als Sprungziel für » Hyperlinks dient.

 Allgemein
 Tag:  <a>
 Icon:  
 XHTML-Version:  1.0
 HTML-Version:  2.0

 Attribute
 name="Name des Ankers"  Name 

 Beispiel
<a name="Oben">Willkommen</a>... Jede Menge Text ...<a href="#Oben">Zurück</a>


Um auf Anker auﬂerhalb der aktuellen Datei zu verweisen, verwenden Sie folgendes Schema: <a href="Seite1.html#Anker">.
