Thumbnail - Generator

Mit diesem Tool können Sie Vorschaugrafiken, sogenannte Thumbnails, erstellen. Der Generator verkleinert eine Gruppe von Grafiken (zum Beispiel Fotos) und verteilt Sie auf mehrere Seiten, durch die dann navigiert werden kann. Das verringert die Ladezeiten und ist übersichtlicher.

 Quellverzeichnis 
Verzeichnis, das die Grafiken (nur *.jpg bzw. *.jpeg) enthält, die verkleinert werden sollen.

 Aufteilung 
Bestimmt die Aufteilung der einzelnen Seiten.

 Ausgabehöhe 
Höhe, auf die die Grafiken verkleinert werden sollen.

 Ausgabebreite 
Breite, auf die die Grafiken verkleinert werden sollen.

 Dateiname(n) 
Bestimmt die Namen, unter denen die einzelnen Seiten gespeichert werden sollen. Das "#" wird dann durch 1, 2, ... ersetzt.

 Thumbnail - Vorlage 
Das hier gewählte Dokument wird als Vorlage für die Thumbnail - Seiten verwendet. Die Vorlage muß die Variable "#Cursor" enthalten. An dieser Position wird dann die Tabelle mit den Grafiken eingefügt.

 Einzelbild - Vorlage 
Die gewählte Vorlage wird für die einzelnen Grafiken verwendet. Die Vorlage muß die Variable "#Cursor" enthalten. An dieser Position wird die Grafik eingefügt.

 Zielverzeichnis 
Verzeichnis, in das alle Dateien (Grafiken, Seiten) kopiert werden sollen.

 Startseite 
Dokument, auf das von jeder Seite aus verlinkt wird ("Home" - Seite).

 Beschriftungen 
Verschiedene Informationen zur Grafik.

 Filter 
Verschiedene Filter, die bei der Größenänderung eingesetzt werden. "Triangle" ist der schnellste Filter, dafür liefert "Mitchell" genauere Ergebnisse.

 Qualität 
Bestimmt die Qualität der Grafiken. Je kleiner der Qualitätswert, desto kleiner ist allerdings auch die Größe der Datei.

Über das Kontext - Menü (rechte Maustaste) der Grafiken - Liste können einzelne Einträge auch wieder entfernt werden.
