Auswahlliste

Erzeugt eine Liste, aus der Einträge gewählt werden können.

 Allgemein
 Tag:  <select>
 Icon:  
 XHTML-Version:  1.0
 HTML-Version:  2.0

 Attribute
 name="Name der Auswahlliste"  Name 
 size="Optische Größe der Liste in Einträgen"  Größe 
 multiple="multiple"  Mehrfachauswahl 

 Beispiel
<form><select size="1"><option>XML</option><option>XHTML</option><option>WML</option></select></form>


Mit dem Attribut size="1" wird eine Drop-Down - Liste angezeigt.
Mehrere Einträge können nur ausgewählt werden, wenn gleichzeitg die Strg - Taste gedrückt wird.
