Multimedia - Objekt

Zeigt eine beliebige Multimedia - Datei an. Dazu gehören Videos, Sounddateien, Texte oder Tabellen.

 Allgemein
 Tag:  <object>
 Icon:  
 XHTML-Version:  1.0
 HTML-Version:  4.0

 Attribute
 data="Multimedia - Datei"  Dateiname 
 standby="Text"  Standby - Text 
 type="Typ der Datei"  Mime - Typ 
 height="Prozent-/Pixelwert"  Höhe 
 width="Prozent-/Pixelwert"  Breite 
 border="Pixelwert"  Randdicke 
 align="top|middle|bottom"  Ausrichtung 

 Beispiel
<object data="tabelle.xls" standby="Bitte warten..." type="application/msexcel" height="100" width="100"></object>


Wenn Sie das Objekt als Verweis einsetzen, verwenden Sie das Attribut border="0", um den Rahmen zu entfernen.
